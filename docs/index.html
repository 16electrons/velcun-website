Quick Start

bash
# 1. Get your API key
# Contact us at hello@velcun.com to get your API key

# 2. Authenticate
curl -X POST https://velcun.com/api/auth/login \
  -H "Content-Type: application/json" \
  -d '{
    "email": "your@email.com",
    "password": "your-password"
  }'

# 3. Use the token in requests
curl -X GET https://velcun.com/api/layers/dashboard \
  -H "Authorization: Bearer YOUR_TOKEN"

Autonomous Settlement API

Automated invoice processing and payment workflow management.

POST
/layers/settlement
Protected
Create new settlement record for automated processing
Request Body
json
{
  "fleetId": "string",
  "loadId": "string (required)",
  "invoiceNumber": "string",
  "carrier": "string",
  "shipper": "string",
  "amount": "number (required)",
  "dueDate": "date (ISO format)",
  "invoiceDate": "date (ISO format)",
  "status": "string (default: pending)",
  "documents": ["array of document objects"],
  "ediData": "object",
  "notes": "string"
}
Response
json
{
  "success": true,
  "message": "Settlement created and processing initiated",
  "id": "string (settlement ID)"
}
GET
/layers/settlement
Protected
Fetch settlement records with optional filtering
Query Parameters
text
?fleetId=string
&status=pending|processing|validated|paid
&startDate=ISO-date
&endDate=ISO-date
&limit=number (default: 50)
&offset=number
Response
json
{
  "success": true,
  "data": ["array of settlement objects"],
  "totalCount": number
}
